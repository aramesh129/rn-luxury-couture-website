# RN Luxury Couture Website

I designed and built this website for a local couture business to improve their online presence.

## Features
- Responsive design
- Navigation bar
- Contact page
- Business information
- Mobile-friendly layout
- Appointment booking with fucntionality to add events to store owners google calender

## Technologies Used
- HTML  
- CSS  
- Google App Scripts
